## Overview
Screen shots providing detail of desired outcomes for project

## Outcomes
**Device Enrollment - Intune Console View**
Enrolled device appears in Intune portal showing managed status, compliance state, and last check-in timestamp.
![Console](/images/ProofOfConecpt-DeviceIntuneEnrollment(Console).png)

**Device Enrollment - End User View**
Device settings confirm successful enrollment with "Managed by LearningInc" indicator and active sync connection.

![Device](/images/ProofOfConecpt-DeviceIntuneEnrollment(Device).png)

**Configuration Policy Enforcement - BitLocker**
BitLocker encryption policy triggered automatically post-enrollment, prompting user to encrypt device per compliance baseline.
![Bitlocker Notification](/images/ProofOfConcept-BitLocker.png)
 
 
**Application Deployment - Microsoft Whiteboard**
Managed application deployed successfully to TestUsers group, with automatic installation notification displayed to end user.

![Software Distribution Notification on Device](/images/ProofOfConcept-Whiteboard.png)
