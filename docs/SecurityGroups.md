# Security Groups

## Task-Overview
Created target groups to collectively manage policies, monitor compliance status, and deploy configuration baselines for devices and users.

## Purpose
Allows for logical grouping of devices and/or users to properly allocate, update, and manage compliance monitoring, configuration baselines, and policy assignment or removal.
